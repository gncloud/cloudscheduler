Features

AWS EC2

We focus only on AWS EC2 instances to provide the best possible service.

All regions

You can see the list at a glance for all instances of the region, and you can also set the schedule.

Any device

Set up a schedule at any time from any device, including mobile.

Up to 70% cost savings

For customers who leave all of their Amazon EC2 instances running 24 hours a day, this solution can result in up to 70% cost savings for those instances that are only necessary during regular business hours.

Flexible schedule setting

You can set schedule by day from Monday to Sunday. If you do not set an stop time, the instance will remain running until the next day.

All region view for EC2 instances

You do not have to go through all the regions of AWS to see the instances. The cloud scheduler can be used as an EC2 instance monitoring page.

Instance Information

You can check all the information of the EC2 instance, and you can turn the schedule on and off with one click. It also provides links to the AWS EC2 instance information page.

Keys

Safe and convenient to manage instances based on AWS keys with limited permissions.
